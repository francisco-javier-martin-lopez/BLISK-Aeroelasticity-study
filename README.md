# BLISK-Aeroelasticity-study

Worked with various internal software for CFD simulations and meshing, CAD with Catia V5, Autogrid for meshing and Tecplot & Paraview for postprocessing.

Starting with the experimental data of the ACTIVE test rig, replicating the experiments in CFD for various solvers (both linear & non-linear) and comparing both aerodamping and flow fields obtained. 

First presentation contains the linear simulations data, and the second one the firsts results for non-linear simulations, with different iterations of the mesh and domain modifications.
